# SWAP2YUNTA
SWAPIDC对接云塔IDC系统插件
## 安装教程
1. 下载之后把`XiaoYu_YunTa.php`和`version.php`放到`/swap_mac/swap_lib/server/XiaoYu_YunTa`里面
2. 检查是否安装完成，如果不行请检查目录。
